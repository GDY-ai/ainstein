- **技术栈与构建**：采用 `Vite` 作为构建工具，配合 `@vitejs/plugin-react` 支持 React 开发，使用 `TypeScript` 进行类型约束。
- **路由与入口**：`src/main.tsx` 为应用入口，集成 `react-router-dom` 并配置 `basename="/ainstein"`；`src/App.tsx` 定义全局路由表，包含登录、大脑列表/详情/创建等核心路径。
- **状态与鉴权**：通过 `src/api.ts` 封装统一的 `request` 方法，利用 `localStorage` 持久化存储 Token 和用户信息，并在 `App.tsx` 中通过 `RequireAuth` 组件实现路由级访问控制。
- **数据层设计**：`src/types.ts` 集中定义业务实体接口（如 `Brain`, `KnowledgeGraph`, `User`），`src/api.ts` 暴露语义化的 API 调用方法（如 `listBrains`, `getKnowledgeGraph`），实现视图层与后端接口的解耦。
- **页面结构**：`src/pages/` 目录下按功能划分页面组件（如 `BrainList.tsx`, `BrainView.tsx`），直接消费 API 模块提供的数据服务。
