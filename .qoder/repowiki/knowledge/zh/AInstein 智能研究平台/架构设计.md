- **单体全栈架构**：采用 Flask 作为核心运行时，同时承载 RESTful API 后端逻辑与 React 前端静态资源托管，通过 `/ainstein` 路径前缀实现路由隔离。
- **共享数据契约**：前后端及智能体模块均依赖统一的 SQLite 数据层（`database.py`），该层不仅管理传统项目实体，还定义了硅基大脑（Brains）、认知元素（Cognitive Elements）及事件总线（Events）等复杂图谱结构，作为跨模块通信的唯一事实来源。
- **异步任务解耦**：后端通过线程池（`threading.Thread`）将耗时的智能体推理任务（如 `run_research_session`）从 HTTP 请求周期中剥离，确保前端界面的响应实时性，并通过数据库状态轮询或事件机制同步执行进度。
