- **事件驱动架构**：以 `EventBus` 为中枢神经，连接前端交互、后端业务逻辑与智能体行为，实现解耦的异步通信。
- **ATA编排机制**：`ATAOrchestrator` 作为心跳引擎，监听事件并调度 `AgentPool` 中的智能体进行思考循环，通过指数退避算法优化资源消耗。
- **认知图谱演化**：`cognitive` 模块维护动态知识图谱，通过置信度更新与矛盾检测自动触发 `DeliberationEngine` 进行多轮博弈，推动知识收敛。
- **前后端分离契约**：前端通过 RESTful API 与后端交互，利用 TypeScript 类型定义确保数据结构一致性，支持实时图谱可视化与大脑状态监控。
