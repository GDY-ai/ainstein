- **事件驱动编排**：后端以 `ATAOrchestrator` 为核心，通过 `EventBus` 监听认知元素（CE）创建、博弈结束等事件，动态唤醒对应大脑的思考循环并调度 Agent 执行任务。
- **认知状态同步**：前端通过轮询 `/knowledge-graph` 和 `/observer-logs` 接口，利用 D3.js 实时渲染由后端博弈产生的动态知识图谱及观察员总结，实现人机协同的认知探索。
- **模块化边界**：后端将 Agent 逻辑（`agents/`）、博弈引擎（`deliberation.py`）与认知存储（`cognitive.py`）解耦，通过统一的 API 层（`app.py`）向前端暴露原子化的认知操作能力。
