- 所有页面组件均采用内联 CSS 对象（`CSSProperties`）进行样式定义，未使用 CSS Modules 或预处理器。
- API 请求统一通过 `src/api.ts` 中的 `request` 包装器发起，自动处理 `Authorization` 头注入及 401 未授权状态的本地存储清理。
- 路由权限控制采用高阶组件模式，在 `App.tsx` 中通过 `RequireAuth` 包裹受保护的路由元素，检查 `getToken()` 返回值。
- 类型定义集中管理在 `src/types.ts` 中，涵盖业务实体（Brain, User）及 API 响应结构，并在组件和 API 层广泛引用。
