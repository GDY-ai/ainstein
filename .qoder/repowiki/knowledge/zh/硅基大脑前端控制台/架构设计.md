- **技术栈**: 采用 React 18 + TypeScript + Vite 5，使用 `react-router-dom` 进行客户端路由管理。
- **入口与路由**: `src/main.tsx` 为应用入口，挂载 `BrowserRouter`；`src/App.tsx` 定义核心路由结构，包含登录页 (`/login`)、大脑列表 (`/brains`)、大脑详情 (`/brain/:id`) 等，并通过 `RequireAuth` 组件实现受保护路由。
- **数据层**: `src/api.ts` 封装了统一的 `request` 函数，处理 JWT Token 的自动注入、401 状态下的本地存储清理以及错误解析；`src/types.ts` 定义了全量 TypeScript 接口（如 `Brain`, `User`, `KnowledgeGraph`）。
- **视图层**: `src/pages/` 目录下存放页面级组件（如 `Login.tsx`, `BrainList.tsx`），采用内联 CSS 样式对象（Inline Styles）实现 UI，无外部 CSS 文件依赖。
- **状态管理**: 依赖 `localStorage` 持久化用户 Token 和信息，通过 `api.ts` 中的工具函数进行读写，未引入 Redux/Zustand 等全局状态库。
