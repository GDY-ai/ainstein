- 所有 API 请求统一通过 `src/api.ts` 中的 `request` 辅助函数发起，自动拼接 `/ainstein/api` 前缀并处理非 200 状态码异常。
- 业务实体类型（如 Project, Session, Finding 等）集中定义在 `src/types.ts` 中，并在组件中通过 `import type` 引入以保持类型安全。
- 页面组件普遍采用“局部状态 + useEffect 触发加载”的模式，通过 `useState` 管理数据与 UI 状态，`useEffect` 处理初始化或依赖变更时的数据获取。
- UI 样式主要采用内联 CSS 对象（`React.CSSProperties`）结合 CSS 变量（如 `var(--accent)`）的方式实现，未引入额外的样式预处理器或 CSS-in-JS 库。
